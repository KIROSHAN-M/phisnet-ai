## 🧠 About the Project

💡 **PhishNet-AI** is an intelligent AI-based system that detects phishing attacks in real time.

🔍 It analyzes:
- URLs
- Emails
- Website patterns

⚡ Provides fast and accurate threat detection.

---

## 🎯 Features

✨ Smart URL Detection  
✨ AI-based Classification  
✨ Real-time Threat Analysis  
✨ Clean & User-friendly UI  
✨ Secure and Scalable System

## ⚙️ Installation

```bash
git clone https://github.com/KIROSHAN-M/phisnet-ai.git

cd frontend
npm install
npm run dev

cd ../backend
pip install -r requirements.txt
python app.py
```
